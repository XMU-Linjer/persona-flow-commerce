# Shop Agent

店铺经营决策 Agent 项目。

## 项目目标

通过业务数据分析、规则系统和 AI 能力，为店铺经营者提供可执行的经营建议。

## 项目结构

- `backend`：Spring Boot 后端服务
- `frontend`：Web 前端
- `ai-service`：Python AI 服务
- `infra`：MySQL、Redis、RabbitMQ 等基础设施配置
- `docs`：架构、接口和项目说明文档

## 技术栈

- Java 17
- Spring Boot 3.5.x
- Node.js LTS
- Python 3.11
- MySQL 8.4
- Redis 7.4
- RabbitMQ 4.x
- Docker Compose
